# Task Management System

A full stack Task Management application developed using:

- React.js
- Spring Boot
- MySQL
- REST API

## Features

- Add Task
- Update Task
- Delete Task
- Modern UI Design

## Technologies Used 

### Frontend
- React.js
- CSS
- Axios

### Backend
- Spring Boot
- Spring Data JPA
- Maven

### Database
- MySQL

## API Endpoints

### Get All Tasks
GET

### Add Task
POST
/api/tasks

### Update Task
PUT
/api/tasks/{id}

### Delete Task

/api/tasks/{id}
